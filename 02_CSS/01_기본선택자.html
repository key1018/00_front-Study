기본선택자

선택자(Selector)란?

특정 HTML요소를 선택하고자 할 때 사용하는 기능
해당 요소를 선택해서 원하는 "스타일"과 "기능"을 적용시킬 수 있음

1. 모든(전체) 선택자 : *

- 현재 문서상에 존재하는 모든 요소들을 다 선택하고자 할 때 사용


        * {
            스타일속성: 값;
        }

2. 태그 선택자 : 태그명

현재 이 문서상에 해당 태그를 다 선택하고자 할 때 사용

        태그명(ex: h3, p, pre 등등) {
            스타일속성: 값;
        }

3. 아이디 선택자 : #아이디명

아이디란? - 현재 문서상에 특정 요소 "하나만" 선택하고자 할 때 사용하고자 할 때 id=xxx로 명시함
단, 해당 그 요소에 id속성을 이용해서 고유한 아이디값을 부여해야됨
=> 고유한 값이므로 중복불가능 

        #아이디명 {
            스타일속성: 값;
        }

4. 클래스 선택자 : .클래스명

class란?
- 해당 문서상에 내가 원하는 요소를 "여러개" 선택하고자 할 때 사용
해당 선택하고자하는 요소에 class속성값을 부여해둬야됨
=> 중복가능 

        .클래스명 {
            스타일속성: 값;
        }
