1. 속성 선택자

선택하고자하는 요소내에 작성되어있는 속성들을 이용해서 선택
선택자 뒤에 []를 이용해서 속성과 속성값을 제시하면서 선택 

        선택자[속성=속성값] {
        선택자[속성~=속성값] {
        선택자[속성|=속성값] {
        선택자[속성^=속성값] {
        선택자[속성$=속성값] {
        선택자[속성*=속성값] {
    
            스타일속성: 값;
        }

2. 자손선택자와 후손선택자

요소들이 중첩되서 작성 가능
자손 : 바로 하위인 요소들 / 후손 : 하위요소들 전부 

        * 자손선택자
        a > b { 
            스타일속성: 값;
        }

        * 후손선택자
        a b { 
            스타일속성: 값;
        }

3. 동위(같은레벨) 선택자

동위 관계에 있는 뒤에 위치한 특정 요소를 선택할 때 사용

        * a요소 "바로 뒤"에 있는 b요소 "하나만"을 선택
        a + b {
            스타일속성: 값;
        }

        * a요소 뒤에 "모든" 요소들 다 선택 
        a ~ b {
            스타일속성: 값;
        }

4. 반응 선택자

사용자의 움직임에 따라 선택되는 선택자

        * 특정 요소에 클릭이 되는 순간 스타일을 부여하고자 할 때
          선택자(태그, 아이디, 클래스 등등):active {
                스타일속성: 값;
          }

        * 특정 요소에 마우스가 올라가는 순간 스타일을 부여하고자 할 때
         선택자:hover{
                스타일속성: 값;
         }
active테스트

link 및 visited 요소

        * 링크 요소들 중 방문하지 않은 사이트의 경우 보여지는 스타일을 부여하고자 할 때
        선택자:link {
            스타일속성: 값;
        }

        * 링크 요소들 중 방문한 사이트의 경우 보여지는 스타일을 부여하고자 할 때
        선택자:visited {
            스타일속성: 값;
        }

5. 상태선택자

요소의 상태에 따라서 선택되는 선택자

        1) 체크(checked)된 상태의 요소에 스타일 부여하고자 할 때 => 체크해제되면 다시 원상복귀됨
           선택자:checked {
               스타일속성: 값;
           }
    
        2) 초점(focus : 커서가 깜빡일때)이 맞춰진 input요소에 스타일을 부여하고자 할 때 
           선택자:focus {
                스타일속성: 값;
           }

    3) 활성화 되어있는(enabled : 사용가능한 상태) 요소에 스타일을 부여하고자 할 때
       선택자:enabled {
           스타일속성: 값;
       }
    
    4) 비활성화 되어있는(disabled : 사용불가능한 상태) 요소에 스타일을 부여하고자할 때
       선택자:disabled {
           스타일속성: 값;
       }
