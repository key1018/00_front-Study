font-size : 글꼴의 크기를 변경할 때 사용

선택자 { font-size : px | em | % }
  1. 고정크기의 글자(px)
  2. 가변크기의 글자(em)
  3. 가변크기의 글자(%)

font-weight: 글꼴의 굵기를 표현할 때 사용

선택자 { font-weight: bold | bolder | ligher | 100~900; }

  • 원래 굵기
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 가늘게

font-variant : 영문텍스트를 작은 대문자로 변경할 때 사용

선택자 { font-variant:small-caps; }

font-style : 텍스트 문구를 기울여서 표현하고자 할 때 사용

선택자 { font-style:italic|oblique;}

italic 기울임 글꼴(원래부터 기울어져있는 글꼴)

oblique 원래글자를 기울인셈(기울여지지않은 글꼴을 기울이게함)
