텍스트 관련 스타일

color : 텍스트 색상을 지정

        선택자 {
            color: 색상명 | 16진수 | rgb(x,x,x) | rgba(x,x,x,x) | hsl(x,x,x) | hsla(x,x,x,x);
        }

text-decoration : 텍스트에 줄을 긋거나 줄을 없앨 때 사용

        선택자 {
            text-decoration : underline | overline | line-through | none;
        }

text-transform : 영문 텍스트의 대소문자 변환시 사용

        선택자 {   
            text-transform : uppercase | lowercase | capitalize;
        }

text-shadow : 텍스트에 그림자 효과를 줄 때 사용

        선택자 {
            text-shadow : 가로거리(x) 세로거리(y) [번짐정도] [색상];
        }

text-align : 텍스트 정렬할 때 사용

line-height: 줄 간격 조절할 때 사용

            선택자 {
                text-align: left(기본값) | justify(양쪽정렬) | right | center(가운데정렬);
                line-height: px | em | %;
            }
