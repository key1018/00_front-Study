목록 관련 스타일

list-style-type: 불릿기호를 변경시켜줄 때 적용

list-style-image: 불릿기호를 이미지로 변경하고자 할 때 사용

list-style-position: 불릿기호의 위치를 조절할 때 사용

        선택자 {
            list-style-type: disc(채워진원) | circle(빈원) | square(네모) | none(불릿기호없음);
            list-style-image : url('적용시키고자하는 이미지의 경로');
            list-style-position : inside(기존보다 더 안으로 들어감) | outside(기본값);  
        }
