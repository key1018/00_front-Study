jQuery 개요

* jQuery 란?

기존의 자바스크립트 언어만으로 구현하기에 복잡했던 구문 작성들을
간소화하기 위해서 "존 레식"에 의해 만들어짐
JavaScript 기반 라이브러리 (유용한 기능들의 모음집)
즉, JavaScript 코드로 유용한 함수나 내용들이 이미 정의되어있는 모음집

>> jQuery의 장점
1) 복잡한 자바스크립트 구문을 보다 짧고 쉽게 개발 가능
2) AJAX(비동기식 통신), 이벤트 처리 등등 폭넓게 지원함
3) jQuery와 관련된 확장형 플로그인, 오픈 api등을 지원함 (차트, 슬라이드, 애니메이션 등등 .. )

* jQuery 라이브러리 연결 방법

head 태그 내에 script 태그로 외부 jQuery관련 .js 파일 연결하기
단, 오프라인/온라인 방식 있음
> jQuery 공식 사이트
jQuery 공식 사이트로 이동

1. jQuery 라이브러리 직접 다운로드 받은 후 경로 지정 (오프라인 방식)
2. CDN(Content Delivery Network) 방식 (온라인 방식)
파일을 다운로드 받아서 저장할 필요 없이 라이브러리를 제공하는 사이트 url제시해서 파일 경로 지정

* jQuery 구문 작성하기

jQuery 구문으로 dom요소를 다루는 구문을 기본적으로 작성할 것이기 떄문에
문서상에 요소들이 다 만들어지고나서 실행되어야만 함

>> 순수 자바스크립트 에서는 

            window.onload = function() {
                실행시키고자하는 내용;
            }
            
            window.onload는 단 한 번만 작성가능


>> jQuery 에서는 3가지 구문으로 제공 
            1) jQuery(document).ready(function() {
                    실행시키고자하는 내용;
               });

            2) $(document).ready(functionc() {
                    실행시키고자하는 내용;
               });

            3) $(function() {
                    실행시키고자하는 내용;
               });

            여러번 기술 가능(제한없음)
            단, 작성되어있는 순서대로 실행됨
